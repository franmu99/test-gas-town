# RentaCar - App de alquiler de coches

Aplicación web de alquiler de coches estilo Kayak/Rentalcars, construida con Next.js y Tailwind CSS.

## Tech Stack

- **Next.js 16** (App Router)
- **Tailwind CSS 4** (paleta azul/slate)
- **TypeScript**
- **Heroicons**
- **date-fns**

## Estructura

```
app/            → Rutas y layouts (App Router)
  (pages)/      → Grupo de rutas para páginas
components/     → Componentes reutilizables
lib/            → Utilidades y helpers
data/           → Datos estáticos y tipos
```

## Desarrollo

```bash
npm run dev     # Servidor de desarrollo
npm run build   # Build de producción
npm run start   # Servidor de producción
npm run lint    # Linter
```
